Kinext - Making it easy for physiotherapists

Kinext is an application we build with the company Feel-IO where I'm co-owner of. Kinext is an application that makes it easy for physiotherapists to manage there patients and what exercises they need to do at home. There is also a small portal where a patient can see there own exercises with videos and/or images.

Mollie Client - A Java client build on top of the Mollie Rest API

We are using the Mollie payment platform in our Kinext application. As our back-end for Kinext is written in Java we needed an easy way to talk to the Rest API from Mollie. That's why we decided to create a client for this. Because this client can be useful to other developers using the Mollie payment platform we decided to make it opensource.
